Module: jsx

RegisterPage component for user registration. Users can register with email/password or via Google authentication. Displays loading states and handles errors during the registration process.

(inner) handleGoogleRegister() → {Promise.<void>}

Handles the Google registration process. It allows users to sign in using their Google account and send the Google ID token to the backend for verification. Upon successful registration, the user is redirected to the login page. In case of an error, an error message is displayed.

Returns:

This function does not return anything. It handles asynchronous operations for Google-based user registration.

Type
Promise.<void>

(inner) handleSubmit(e) → {Promise.<void>}

Handles the form submission for user registration. It sends a POST request to create a new user and verifies the user's email. Upon successful registration, it redirects the user to the login page. In case of an error, an error message is displayed.
